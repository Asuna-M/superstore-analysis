1. Цель проекта

Анализ датасета продаж супермаркета для выявления:
- Временных трендов продаж и прибыли
- Географических паттернов эффективности
- Влияния скидок на конечную прибыль
- Оптимизации продуктовой стратегии

2. Используемые инструменты

Power BI:
- Power BI Desktop — основная программа
- Power Query — инструмент для обработки дата сета
- DAX (Data Analysis Expressions) — расчётные метрики
   
С помощью DAX было написано 2 выражения для расчета ключевых метри и последуещего их использования в графическом виде

Рентабельность = 
DIVIDE(
    SUM('Superstore'[Profit]),
    SUM('Superstore'[Sales]),
    0
) * 100

Средний чек = 
DIVIDE(
    SUM('Superstore'[Sales]),
    SUM('Superstore'[Quantity]),
    0
)

4. Аналитические выводы и бизнес-рекомендации

На основании моего анализа представляю некоторые теоретические рекомендации 

1.
Пересмотреть скидочную политику в штатах
Как пример:
Texas: снизить максимальную скидку до 25%
Ohio: ввести скидки только для постоянных клиентов
Pennsylvania: заменить скидки на бонусные программы

2.
Внедрить сегментацию клиентов:
Разделить на "ценовых" и "качественных"
Разные ценовые стратегии для разных сегментов

Запустить A/B тесты:
Тест 1: Снижение скидок на 10% в Texas
Тест 2: Премиальные товары без скидок

3. 
Попробовать убрать некоторые сегменты товаров из регионов
Например для штата Техас оставить один сегмент - Технологии, так как только он приносит прибыль
Аналогично пересмотреть штаты с максимальынм убытком.
